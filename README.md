# cryptologykey_claude

Slash-команды (skills) для Claude Code — система автоматизации контента и маркетинга Cryptology Key.

## Воронки и рассылки

| Команда | Описание |
|---------|----------|
| `/webinar-funnel` | Промо-воронка вебинара: анонс → эфир → оффер → дожим. 25 MSG, 7+ каналов, RU+UA |
| `/free-course-funnel` | Воронка бесплатного курса: 5 блоков (вход → уроки → CTA → ре-энгейдж → VSL). 8 курсов-эталонов |
| `/okx-mailing` | Брендовые рассылки OKX: 6 типов, 4 канала (TG Official, боты, Email RU/UA) |
| `/test-drive-funnel` | Воронка Test Drive / Open Days: Workshop TD, FTT OD, Intro OD. 27 MSG, 20 углов офферов |
| `/funnel-messages` | Сообщения воронки (TG-бот + email) |
