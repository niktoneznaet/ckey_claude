## Задача

Создай полный набор промо-сообщений для вебинара CK: от анонса (день -1) до пост-вебинарного дожима (день +1). Это НЕ сценарий самого вебинара (для этого есть /webinar-script), а **промо-воронка вокруг вебинара** — все сообщения которые отправляются по всем каналам.

## Входные данные

Запроси (если не указано):

1. **Тема вебинара** — о чём эфир
2. **Спикер(ы)** — кто ведёт (обычно GZK + один из: Garry, Solva, D-Trader, Nana)
3. **Дата и время** — когда эфир (по умолчанию 19:30)
4. **Продукт для оффера** — что продаём после вебинара:
   - `FTT` — Full-Time Trader (флагман, 2+2 месяца)
   - `Workshop` — повышение квалификации (11 недель)
   - `Alpha Trader` — психология торговли (8 модулей, Nana)
   - `Workshop TD` — бесплатный Test Drive (5 дней)
   - `без оффера` — чисто контентный эфир (CTA на соцсети)
5. **Бонусы предзаписи** (если есть) — скидка, доп. курс, расширенный доступ

**[TRIGGER]** Одновременно с MSG-001

**[ФОРМАТ]** Письмо от имени спикера

**⚠️ ПРАВИЛА EMAIL (из email-deliverability-guide.md):**
- From name: имя спикера (Garry / Solva), НЕ "Cryptology Key"
- Тема: 30-50 символів, без spam-тригерів (без "безкоштовно", "акція", CAPS, emoji)

- Без списків 5+ пунктів (максимум 3-4)
- HTML до 102 KB, банер максимум 40%
- Plain text версія обов'язкова

Структура:
```
From: [Ім'я спикера]
Тема: [30-50 символів, формула: питання / число+конкретика / від першої особи / протиріччя]

[Особистий контекст від першої особи — 2-3 речення. Нерівний ритм. Конкретика, не абстракції. Як пишеш другу.]

— [пункт 2]
— [пункт 3]

[CTA кнопка: Зареєструватися] → одна кнопка, один CTA

[Ім'я]
```

**[ЗАМЕТКА]** Email ЗАВЖДИ українською. Стиль — особистий лист, без верстки, без зображень (або мінімум). Перевір тему на spam-тригери перед відправкою.

**ПРАВИЛА ОФФЕРА (из стиля CK):**
- НЕ "покупай" → "забронируй место" / "делай предзапись"
- Привязка к теме вебинара (то, что разбирали → почему продукт это решает)
- Антипродажность: "Хочешь — купи, не хочешь — не купи"
- НЕ обещать конкретных цифр дохода
- НЕ использовать "лёгкий путь", "быстрый результат"
- Можно использовать метафоры CK (борщ, рыбалка, рельсы, аэропорт)

Структура:
```
[Хук-переход от темы вебинара к продукту — 1-2 предложения]

[Название продукта] — [одно предложение что это]

— [выгода 1]
— [выгода 2]
— [выгода 3]
— [выгода 4]

[Бонусы предзаписи, если есть]
[Дедлайн, если есть]

[кнопка 1: Записаться на курс → ссылка + UTM]
[кнопка 2: Хочу консультацию → ссылка]
```

- 2 месяца теории + 2 месяца практики с менторами
- 4 уровня: Junior / Middle / Senior / SuperCharge
