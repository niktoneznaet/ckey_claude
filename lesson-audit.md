# Аудит урока / контента курса

## Задача

Проведи аудит готового урока (текст суфлёра, транскрибация видео, или скринкаст) на соответствие методологии, качество подачи и эффективность для целевой аудитории.

## Входные данные от пользователя

1. **Текст урока** — транскрибация или текст для суфлёра
2. **Номер и позиция урока** в курсе (первый / середина / финальный)
3. **Название курса** и общая тема

## Критерии аудита

### 1. Структура урока
- [ ] Хук в первые 15–30 секунд (цепляет?)
- [ ] Контент: информация + мотивация + трансформация
- [ ] Закрытие: резюме + превью следующего + CTA
- [ ] Хронометраж 5–15 минут

### 2. Качество контента
- [ ] Соответствует позиции в курсе (первый «продаёт», середина = ценность, финал = эмоции)
- [ ] Есть быстрая победа / конкретное действие
- [ ] Баланс сложности (не перегружен)
- [ ] Смена форматов каждые 2–3 минуты
- [ ] Конкретика вместо абстракций

### 3. Соответствие аватару (см. CLAUDE.md)
- [ ] Закрывает боли ЦА
- [ ] Язык понятен новичку (нет необъяснённого жаргона)
- [ ] Ведёт к желаемому результату

### 4. Соответствие позиционированию бренда
- [ ] Тон: честный наставник, не «успешный успех»
- [ ] Нет пустых обещаний
- [ ] Есть реальные примеры / истории

### 5. Технические моменты
- [ ] Текст пригоден для суфлёра (короткие предложения, пунктуация)
- [ ] Нет дублей, опечаток, рваных фрагментов
- [ ] Пометки для экрана/слайдов на месте

## Формат выхода

```
# Аудит Урока N — [Название]

## Общая оценка: X/10

## ✅ Что работает отлично
(по пунктам с цитатами из текста)

## ⚠️ Что нужно поправить
(конкретные рекомендации + предложенные формулировки)

## 🔴 Критические проблемы (если есть)

## Ключевые фразы для переиспользования
(сильные формулировки, которые стоит сохранить)

## Рекомендации для следующего урока
(что учесть исходя из того, что было в этом)
```
