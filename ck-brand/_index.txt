## Как применять в новых презентациях

### 1. CSS variables — через tokens.css

```html
<link rel="stylesheet" href="path/to/tokens.css">
<style>
  body {
    background: var(--bg-black);
    color: var(--text-primary);
    font-family: var(--font-mono);
  }
  .accent { color: var(--accent-lime); }
  .card   { background: var(--surface-1); border: 1px solid var(--border-subtle); }
</style>
```

### 2. Master frame (slide chrome)

Каждый слайд 1920×1080, 16:9, чёрный фон + dotted pattern (#1F1F22, шаг 24px) + inner border (отступ 28-32px) + CK wordmark top-left + footer slots bottom (label slева, page X/Y справа).

### 3. Typography

JetBrains Mono. 9 ступеней (от 152px Display XL до 13px Caption). Веса: 700 (Bold), 500 (Medium), 400 (Regular), 200 (thin nums).

### 4. Color rules

- **Один акцент на слайде** — обычно lime, изредка semantic (orange для warn, red для alert).
- **Tier colors** (Junior/Middle/Senior/Supercharged) — только для tier-карточек, не для произвольных акцентов.
- **Logobook palette** (Purple/Ice/Midnight/Emerald) — НЕ для слайдов школы. Только корпоративка/partner-pitches.

### 5. Принципы grammar (см. секцию 03)

- Monospace везде
- Pure black (#000000)
- Outlined frame (1px #2A2A2E)
- Прямоугольные cards (border-radius 0)
- Outlined thin nums (weight 200, stroke 1px)
- CK wordmark в углу каждого слайда
