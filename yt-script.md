## Общая философия

Видео строится на формуле: **Личная история + Структурированная система + Практические шаги**

Зритель должен:
1. Узнать себя в проблеме
2. Поверить, что ты её решил
3. Получить чёткий план действий
4. Захотеть больше (подписка, следующее видео, продукт)

---

## СТРУКТУРА ВИДЕО (12 блоков)

### Блок 1: ХУК (0:00 — 0:30)

**Задача:** Зацепить за 5 секунд, показать ценность видео.

**Формула:**
```
[Провокационный вопрос/утверждение] + [Обещание конкретного результата]
```

**Шаблоны:**
- «Всем привет. В этом видео мы разберём, как [результат]. Я дам вам [X] шагов/принципов, которые...»
- «Простой вопрос: [вопрос, который заставляет задуматься]?»
- «Большинство людей думают, что [общепринятое мнение]. Но это не так.»

---

### Блок 2: ТВОИ ДОСТИЖЕНИЯ (0:30 — 1:00)

**Задача:** Доказать, что тебя стоит слушать.

**Формула:**
```
[Кто ты] + [Конкретные цифры/факты] + [Почему это релевантно теме]
```

**Банк достижений:**
- Cryptology Key: 15 000+ выпускников с 2021
- Смена жизни: стройка/кухня/завод → удалённая работа
- География: Украина → Польша → Варшава
- Трейдинг с 2019, активно с 2021

---

### Блок 3: ОБЕЩАНИЕ ЦЕННОСТИ (1:00 — 1:30)

**Задача:** Чётко сказать, что зритель получит.

**Формула:**
```
«В этом видео я поделюсь [X] [принципами/шагами/правилами], которые [результат].»
```

**Важно:** Всегда называй ЧИСЛО. Число должно быть в названии видео. Число создаёт ощущение структуры и завершённости.

---

### Блок 4: ПРОБЛЕМА — «ОБЫЧНЫЙ ЧЕЛОВЕК» (1:30 — 3:00)

**Задача:** Зритель должен узнать себя.

**Приёмы:**
1. **Персонаж с именем** — «Представьте Галю, у неё двое детей, бизнес по одежде...»
2. **Обобщение** — «Большинство людей просыпаются и первым делом...»
3. **Прямое обращение** — «Было у вас такое, что вы давали себе обещание, а потом...»

---

### Блок 5: КОНТР-ИНТУИТИВНЫЙ ТЕЗИС (3:00 — 4:00)

**Задача:** Сломать ожидания, создать интригу.

**Формула:**
```
«Я не верю в [общепринятое мнение]. Я пришёл к тому, что [парадоксальный вывод].»
```

**Примеры:**
- «Я не верю в дисциплину» (в видео про дисциплину!)
- «Листать рилсы тяжелее для мозга, чем работать»
- «3 часа работы лучше 12»

---

### Блок 6: ТВОЯ ИСТОРИЯ ТРАНСФОРМАЦИИ (4:00 — 5:30)

**Задача:** Эмоциональная связь + доказательство, что система работает.

**Формула:**
```
[Точка А: как было плохо] → [Что понял/сделал] → [Точка Б: как стало]
```

**Важно:** Истории должны быть КОНКРЕТНЫМИ — с датами, местами, цифрами, деталями.

---

### Блок 7: НУМЕРОВАННЫЕ ПРИНЦИПЫ/ШАГИ (5:30 — 20:00+)

**Это ядро видео — 60-70% хронометража.**

**Структура каждого принципа:**
```
1. НАЗВАНИЕ ПРИНЦИПА (чёткое, запоминающееся)
2. ОБЪЯСНЕНИЕ сути (1-2 предложения)
3. ПРИМЕР из жизни / история
4. КАК ПРИМЕНИТЬ (конкретное действие)
```

**Сколько принципов:**
- Короткое видео (10-15 мин): 5-7 принципов
- Среднее (15-25 мин): 7-10 принципов
- Длинное (25-40 мин): 10-12 принципов

---

### Блок 8: МИНИ-ИСТОРИИ ВНУТРИ ПРИНЦИПОВ

**Типы историй:**
1. **Личный провал** — «Когда-то я пытался... и вот что вышло»
2. **Личный успех** — «Я сделал так... и это сработало»
3. **История другого человека** — «Мой друг/ученик/знакомый...»
4. **Метафора** — «Это как если бы ты...»
5. **Контраст** — «Один человек делает так, другой — иначе. Результат...»

---

### Блок 9: НАУЧНЫЕ ВСТАВКИ (опционально)

**Когда использовать:** тема про мозг, привычки, продуктивность — нужно объяснить «почему это работает».

**Формула:**
```
«Есть такая штука — [термин]. Это [простое объяснение]. Простыми словами: [ещё проще].»
```

---

### Блок 10: РЕЗЮМЕ (последние 2-3 минуты)

**Формула:**
```
«Итого, давайте подытожим. [Тезис темы]. Для этого у нас есть [X] принципов:
Первое — [название]. [1 предложение суть]
Второе — [название]. [1 предложение суть]
...»
```

**Важно:** Резюме БЫСТРОЕ — по 1 предложению на принцип.

---

### Блок 11: МОСТ К ПРОДУКТУ/ТЕМЕ (за 1-2 мин до конца)

**Для воронки (софт-скиллы → трейдинг):**
> «И кстати, все эти принципы я активно использую в [своей работе / обучении трейдингу / работе с учениками]. Если вам интересна тема [X], у меня есть [видео/канал/курс]...»

**Или мягче:**
> «А если у вас сейчас стоит задача [связанная тема], очень рекомендую посмотреть моё видео про [X], ссылка будет в описании.»

---

### Блок 12: CTA — ПРИЗЫВ К ДЕЙСТВИЮ (последние 30 сек)

**Стандартный набор:**
1. Лайк
2. Подписка
3. Комментарий (лучше с конкретным вопросом)
4. Ссылка на следующее видео / Telegram / ресурс

---

## ЧЕКЛИСТ ПЕРЕД СЪЁМКОЙ

- [ ] Есть хук с конкретным обещанием (число + результат)
- [ ] Есть 2-3 твоих достижения для доверия
- [ ] Есть описание «обычного человека» с проблемой
- [ ] Есть минимум 1 контр-интуитивный тезис
- [ ] Есть твоя личная история трансформации
- [ ] Каждый принцип имеет пример/историю
- [ ] Минимум 3 личные истории в видео
- [ ] Истории конкретные (даты, места, цифры)
- [ ] Число в названии видео
- [ ] Есть резюме со всеми пунктами
- [ ] Есть CTA с конкретным действием
