# СИСТЕМА СОЗДАНИЯ ЗАЛЕТАЮЩИХ РИЛСОВ (методология Alto)

> **Версия 1 (архивная).** Актуальная версия курса — в knowledge-base/reels-course/. Эта инструкция сохранена как референс оригинальной методологии Alto.

Формат запроса:
```
ТЕМА/НИША: [О чём блог, что продаёшь]
ЦЕЛЬ РИЛСА: [Охваты / воронка / прямая продажа]
ТИП КОНТЕНТА: [Экспертный / личный / экспертно-продающий / продающий]
ФОРМАТ: [Сторителлинг / динамичный / говорящая голова / подкаст / разбор / лайв]
```

На выходе генерируешь: (1) идею, (2) сценарий с раскадровкой, (3) рекомендации по съёмке.

## ЧАСТЬ 1: СТРАТЕГИЯ И ТИПЫ КОНТЕНТА

### 1.1 Четыре типа контента

Каждый тип контента выполняет свою функцию в воронке продаж:

| Тип | Цель | Доля в контенте |
|-----|------|-----------------|
| **Экспертный** | Охваты, новая аудитория, доверие | 50-60% |
| **Личный** | Лояльность, эмоциональная связь | 20-30% |
| **Экспертно-продающий** | Заявки в воронку, прогрев | 15-20% |
| **Продающий** | Прямые продажи | 5-10% |

**Правило:** Нельзя продавать без прогрева. Сначала экспертный и личный контент, потом продающий.

### 1.2 Воронка продаж через рилсы

```
РИЛСЫ (удочка) → СТОРИС/TELEGRAM (прогрев) → YOUTUBE/ПРОДУКТ (продажа)
```

**Рилсы НЕ продают напрямую.** Они:
- Привлекают холодную аудиторию
- Переводят в воронку (Telegram, рассылка)
- Создают точки касания

## ЧАСТЬ 2: АЛГОРИТМЫ INSTAGRAM

### 2.1 Главные факторы ранжирования

**Глубина просмотра (retention)** — самый важный фактор:
- Первые 3 секунды решают всё
- Досмотры до конца = вирусность
- Пересмотры умножают охваты

**Авторский стиль:**
- Уникальная подача = запоминаемость
- Визуальные якоря (очки, кепка, цвет)
- Узнаваемый формат

### 2.2 Что убивает охваты

- Водяные знаки TikTok
- Низкое качество видео
- Отсутствие хука в первые 3 секунды
- Слишком узкая тема (мало кому интересно)

## ЧАСТЬ 4: ГЕНЕРАЦИЯ ИДЕЙ

### 4.1 Система 5x5x3 = 75 идей

**Шаг 1:** Выбери 5 подтем твоей ниши
**Шаг 2:** Для каждой подтемы используй 5 рубрик
**Шаг 3:** Из каждой рубрики вытащи минимум 3 идеи

**Результат:** 5 x 5 x 3 = 75 идей минимум

### 4.2 Пять рубрик для экспертного контента

| Рубрика | Формула | Пример |
|---------|---------|--------|
| **Как сделать** | "Как [результат]" | "Как похудеть на 10 кг" |
| **Почему** | "Почему [проблема]" | "Почему диеты не работают" |
| **Что** | "Что [делать/не делать]" | "Что есть после тренировки" |
| **Ошибки** | "N ошибок в [теме]" | "5 ошибок при похудении" |
| **Инструменты** | "N инструментов для [результата]" | "3 приложения для подсчёта калорий" |

### 4.3 Пять способов широкой подачи

Узкая тема не наберёт охваты. Расширяй через:

| Способ | Узко | Широко |
|--------|------|--------|
| **Результат** | "Как настроить таргет" | "Как привлечь 1000 клиентов" |
| **Эмоция** | "Как вести учёт финансов" | "Как перестать тревожиться о деньгах" |
| **Боль** | "Как составить контент-план" | "Почему твои рилсы не набирают просмотры" |
| **Аудитория** | "SMM для бизнеса" | "Как мамам в декрете заработать на соцсетях" |
| **Контекст** | "Как я монтирую" | "Как я монтирую рилсы в самолёте" |

### 4.4 Механика контекстов

Контекст делает обычную тему интересной. 6 элементов сильного контекста:

1. **Ограничение** — "за 30 дней", "с нуля", "без бюджета"
2. **Противоречие** — "интроверт стал спикером"
3. **Эмоция** — "после развода", "уволили в 40"
4. **Конкретика** — "247 клиентов", "$11K"
5. **Сравнение** — "я vs конкуренты", "до/после"
6. **Триггер любопытства** — "никто об этом не говорит"

### 4.5 Четыре подводки к воронке

| Подводка | Пример |
|----------|--------|
| **Прямая** | "Пиши + в комментарии" |
| **Через пользу** | "Собрал чек-лист — пиши 'чек-лист'" |
| **Через интригу** | "Как именно — рассказал в видео, пиши 'хочу'" |
| **Через кейс** | "Так Маша заработала 500К — хочешь так же? Пиши 'курс'" |

## ЧАСТЬ 5: БАЗОВАЯ СЦЕНАРНАЯ СХЕМА

### 5.1 Структура любого рилса

```
ХУК (3-5 сек) → АКТУАЛИЗАЦИЯ → ОСНОВНАЯ ЧАСТЬ → ВЫВОД → CTA
```

**Хук:** Зацепить внимание в первые 3-5 секунд
**Актуализация:** Дать причину досмотреть (КРИТИЧНО!)
**Основная часть:** Раскрыть основной смысл
**Вывод:** Подвести итог
**CTA:** Перевести на подписку, воронку, заявку

### 5.2 Три подхода к логике сценария

**1. Мысли вытекают друг из друга**
Каждая мысль плавно переходит в следующую. Зритель не теряет суть.

**2. От частного к общему**
Маленькая деталь → большая тема.
*Пример:* "Попал в больницу из-за бутерброда" → День рождения → Бутерброд → Арахис → Аллергия → Больница

**3. От общего к частному**
Большая тема → детали.
*Пример:* "Часто попадаю в больницу из-за аллергии" → Вчера на ДР друга → Съел бутерброд с арахисом

## ЧАСТЬ 6: ШАБЛОНЫ СЦЕНАРИЕВ

### 6.1 ЭКСПЕРТНЫЙ ТИП

#### Шаблон 1: Варианты решения одной задачи
```
Хук: Боль/вопрос ЦА → Вариант 1 → Вариант 2 → Вариант 3 → Вывод → CTA
```

#### Шаблон 2: Ошибки и причины неудачи
```
Хук: Что не получается → Причина 1 (+решение) → Причина 2 (+решение) → Вывод
```
**Вариации:** причины БЕЗ решений (решение в воронке), одна причина ПОДРОБНО, 8 причин КОРОТКО.

#### Шаблон 3: Раскрытие кейса
```
Хук: Широкая личная тема → Контекст + точка слома → Что конкретно сделал → Результат → CTA
```

#### Шаблон 4: Пошаговый план
```
Хук: Вопрос/история → Шаг 1 → Шаг 2 → Шаг 3 → Результат → CTA
```

#### Шаблон 5: Раскрываем важность (УНИВЕРСАЛЬНЫЙ)
```
Хук → Почему это важно → На что влияет → CTA
```

---

### 6.2 ЛИЧНЫЙ ТИП

**Источники идей:** Прошлое (знакомство, почему выбрал нишу, моменты становления), Настоящее (изменения в жизни, мысли, факты о себе), Будущее (цели, подготовка).

#### Шаблоны:
1. **Как решил проблему:** О проблеме → Как решили
2. **Изменения:** Поменял что-то → Как повлияло
3. **Отчёт:** Начало → Ключевые моменты → Кульминация
4. **Знакомство:** Кто я → Важные детали → Зачем снял видео
5. **Моменты становления:** Хук → Дата 1 + событие → Дата 2 + событие → Что сейчас
6. **Цели:** Озвучить цель → Зачем → Шаги для выполнения

---

### 6.3 ЭКСПЕРТНО-ПРОДАЮЩИЙ ТИП

#### Шаблон 1: Рилс на воронку (продаём тему/нишу)
```
Цепляем внимание → Актуализируем проблему → Даём решение (N принципов) → Итог + результат → CTA
```

#### Шаблон 2: Один элемент системы (ТОП-1 ДЛЯ ВОРОНКИ)
```
Хук → Раскрываем ОДИН элемент + примеры → Результат → "Это 1 из N пунктов моей системы" → CTA: "Пиши [кодовое слово]"
```

#### Шаблон 3: Пошаговый план с математикой
```
Широкий хук: "Если бы я потерял всё..." → Шаг 1-4 (все ПРОСТЫЕ!) → Считаем математику → Кейсы → CTA
```
**Критично:** Шаги должны быть понятны ВСЕМ. Сложные термины = мало просмотров.

---

### 6.4 ПРОДАЮЩИЙ ТИП

#### Шаблон 1: Кейс ученика
```
Хук: Широкая личная тема → Контекст + точка слома → Выбор решения → Что сделал → Результат → CTA
```

#### Шаблон 2: Личный кейс через сторителлинг
```
Точка А → Путь трансформации (даты, события) → Точка Б → Благодаря чему → CTA
```
**ВАЖНО:** Не "успешный успех"! Хорошо: "Отцу пришлось продать машину из-за долгов. Хочу отплатить"

#### Шаблон 3: Презентация продукта
```
Результат, к которому хочет прийти → Шаги (экспертная подача!) → "Как раз это мы делаем на моём продукте" → Результат + кейсы → CTA
```

#### Шаблон 4: Реалити
```
Цель → Ограничения → План → Почему может / не может получиться → "Смотрите в прямом эфире"
```
**5 элементов:** понятная цель, дедлайн, благодаря чему, эмоциональная значимость, ограничения.

## ЧАСТЬ 7: СЪЁМКА И ВИЗУАЛ

### 7.1 Ракурсы (11 видов)

| Ракурс | Когда использовать |
|--------|-------------------|
| **На уровне глаз** | Стандартный, зрительный контакт |
| **Кадр сверху (High Angle)** | Показать размер, убрать шум |
| **Вид с неба (Bird's Eye)** | Визуальный хук, уникальность |
| **Над головой (Overhead)** | Очень редкий, сильная уникальность |
| **Кадр снизу (Low Angle)** | Показать силу, место для текста на небе |
| **Кадр земли (Worm's Eye)** | Небоскрёбы, драматичность |
| **На уровне земли** | Обувь, эмоция "плохо" |
| **Голландский угол** | Странность, динамика |
| **POV (от первого лица)** | Уникальные кадры |
| **Через плечо** | Взаимодействие с объектом |
| **Кадр локации** | Показать где находишься |

### 7.2 Размеры кадра

| Размер | Что видно | Когда |
|--------|-----------|-------|
| **Широкий** | Полный рост + локация | Контекст |
| **Средний** | Фокус на объекте | Основной кадр |
| **Крупный** | 70-80% = лицо | Эмоции, важные мысли |
| **Детальный** | Одна деталь близко | Между кадрами, динамика |

### 7.3 A-roll, B-roll, C-roll

| Тип | Что это |
|-----|---------|
| **A-roll** | Главный кадр, говоришь на камеру |
| **B-roll** | Вставки (архивы, подсъёмки) |
| **C-roll** | Монтажные вставки (инфографика, текст) |

**Правило:** Без B-roll и C-roll ролик скучный — обязательно разбавляй!

## ЧАСТЬ 8: РАСКАДРОВКА

### Три вида раскадровки

| Вид | Когда | Как |
|-----|-------|-----|
| **Текстовая** | Простые съёмки (3-5 кадров) | В скобках в сценарии |
| **Эскизы** | Сложные кадры, снимает другой человек | Рисовать от руки |
| **Детальная** | Очень сложные съёмки, много движений камеры | Детальные рисунки со стрелками |

### Формат текстовой раскадровки

```
"[Текст, который говоришь]"
(Размер кадра, ракурс, что в кадре, где текст)
```

## ЧАСТЬ 10: АЛГОРИТМ РАБОТЫ

### Полный процесс создания рилса

```
1. Определить ЦЕЛЬ (охваты / воронка / продажа)
2. Выбрать ТИП КОНТЕНТА (экспертный / личный / экспертно-продающий / продающий)
3. Выбрать ФОРМАТ (сторителлинг / динамичный / говорящая голова / и т.д.)
4. Сгенерировать ИДЕЮ (система 5x5x3, широкая подача, контексты)
5. Выбрать ШАБЛОН сценария
6. Написать СЦЕНАРИЙ (хук → актуализация → основная часть → вывод → CTA)
7. Сделать РАСКАДРОВКУ (текстовую или эскизы)
8. СНЯТЬ (избегая 4 ошибки)
9. СМОНТИРОВАТЬ
```

### Чек-лист перед публикацией

- [ ] Хук цепляет в первые 3 секунды?
- [ ] Есть актуализация (причина досмотреть)?
- [ ] Тема широкая (интересна многим)?
- [ ] Нет визуального шума?
- [ ] Кадры разнообразные (не одинаковые подряд)?
- [ ] Есть место для текста?
- [ ] Есть CTA в конце?
- [ ] Музыка подходит по вайбу?

## КЛЮЧЕВЫЕ МЕТОДОЛОГИИ ИЗ ВИДЕОУРОКОВ ALTO

### Из урока по форматам (Альтаир)
- 6 актуальных форматов рилсов, каждый разобран по цели + формуле + примерам
- Формат "один элемент системы" — топ для воронки (пример: рилс с очками и кепкой = 1М просмотров, 13К подписчиков)
- Формат "реалити" — 5 обязательных элементов: понятная цель, дедлайн, благодаря чему, эмоциональная значимость, ограничения

### Из урока по идеям (Альтаир)
- Система 5x5x3 для генерации 75+ идей
- Идея = 90% результата рилса
- Рилсы для подписчиков и рилсы для воронок — разные подходы
- Сюжетная линия в блоге может дать 10-30 рилсов из одной темы
- Широкая подача узкой темы — ключ к охватам

### Из урока по сценариям (Альтаир)
- Базовая схема: ХУК → АКТУАЛИЗАЦИЯ → ОСНОВНАЯ ЧАСТЬ → ВЫВОД → CTA
- Актуализация (причина досмотреть) — самый недооценённый элемент
- 3 логики сценария: последовательная, от частного к общему, от общего к частному
- 17+ шаблонов для 4 типов контента

### Из урока по раскадровке и картинке (Альтаир)
- 11 ракурсов, 4 размера кадра, 10 композиций
- A-roll + B-roll + C-roll — обязательная комбинация
- 3 вида раскадровки: текстовая, эскизы, детальная
- 4 главные ошибки: без раскадровки, визуальный шум, одинаковые кадры, нет места для текста

### Из видео Тимочко "Полная схема ведения Reels в 2026"
- За месяц 19 рилсов = 40К подписчиков
