# Рекламный креатив

## Задача

Напиши рекламные текстовые креативы для продвижения курсов / продуктов Cryptology Key. Следуй позиционированию бренда из CLAUDE.md.

## Входные данные от пользователя

Запроси (если не указано):

1. **Что рекламируем** — бесплатный курс / FTT / вебинар / другое
2. **Канал размещения** — Telegram-канал / таргет (Instagram/Facebook) / Google
3. **Язык** — украинский (основной) / русский
4. **Углы подачи** — боль / возможность / формат / все три
5. **Длина** — короткие (3–5 предложений) / средние (абзац) / длинные
6. **Ссылка** — если есть, или поставить плейсхолдер

## Три угла подачи

### 🔴 БОЛЬ
Входная точка — проблема, которую человек уже ощущает.
- Сливает депозит
- Торгует на эмоциях
- Информационный хаос
- Пробовал сам — не вышло

### 🟢 ВОЗМОЖНОСТЬ
Входная точка — мечта, к которой стремится.
- Новая профессия
- Удалённый доход
- Свобода от найма
- Старт с нуля без большого капитала

### 🔵 ФОРМАТ
Входная точка — удобство и доступность.
- Короткие уроки (как рилсы)
- В телефоне / Telegram
- Бесплатно
- За 2 часа уже в теме

## Правила

### Тон (из CLAUDE.md):
- Честный, прямой, без воды
- Не «успешный успех»: без машин, скринов профита, обещаний «$1000 за неделю»
- Можно грубовато, по-человечески
- Конкретика: цифры, сроки, формат

### Структура креатива:
1. **Хук** — первое предложение цепляет (вопрос / утверждение / провокация)
2. **Тело** — 2–3 предложения: суть предложения
3. **CTA** — одно конкретное действие (ссылка)

### Чего НЕ делать:
- Не использовать слово «сигналы»
- Не обещать конкретный заработок как гарантию
- Не использовать emoji-списки на 10+ пунктов (выглядит как спам)
- Не писать «это не очередной bullsh*t» — это уже клише
- Не перечислять все преимущества сразу — один угол на креатив

## Формат выхода

По 2 креатива на каждый запрошенный угол. Каждый креатив:

```
**[УГОЛ] #N**

[Текст креатива]

👉 [CTA + ссылка]
```

После креативов — краткий комментарий: какой из них лучше тестировать первым и почему.
