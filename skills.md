Выведи полный список всех доступных пользовательских slash-команд. Для этого:

1. Просканируй папку ~/.claude/commands/ — это глобальные команды (/user:название)
2. Просканируй папку .claude/commands/ в текущем проекте — это проектные команды (/project:название)
3. Для каждой найденной команды прочитай первые 3-5 строк файла, чтобы понять назначение

Выведи результат в формате:

## Глобальные команды (/user:...)
- `/user:название` — краткое описание (1 строка)

## Проектные команды (/project:...)
- `/project:название` — краткое описание (1 строка)

Если в какой-то категории команд нет — не выводи эту секцию.
