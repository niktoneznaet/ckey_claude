# sites-vova

Шаблонизатор CK-лендингов. Быстрый способ получить рабочий скелет нового лендинга Cryptology Key в фирменном стиле.

## Быстрый старт

1. В Claude Code напиши: `/sites-vova`
2. Ответь на 5-7 вопросов брифа (тип продукта, название, менторы, цена, дата)
3. Получаешь путь к файлу в `outbox/` — открываешь в браузере, видишь готовый скелет.

Пример вызова:
```
/sites-vova
> Free course "Solana Starter", ментор Solva, старт 2026-05-15,
> оффер "Знайди наступну Solana за 5 уроків", аудиторія новачки
```

Выход: `outbox/landing-solana-starter-2026-04-22.html`

## Что внутри

- **3 шаблона** — paid course (эталон FTT), free course (эталон Base Camp), event (эталон OKX Summit / NeKonfa)
- **CK master-токены** — одна палитра, одна типо, один CTA-стиль. Защищает от brand fragmentation.
- **Библиотека компонентов** — mentor-card, pill-CTA со стрелкой ↗, timer, stats-row, FAQ, footer.
- **Reference-анатомия** — по-секционный разбор FTT, Base Camp, OKX Summit (чтобы понять, как система устроена).

## Принципы дизайн-системы CK

- **Master + accent.** Чёрный/белый — основа. Accent цвет — один на продукт. Никаких 8 палитр.
- **CTA-капсула со стрелкой ↗** — фирменный sign, на всех кнопках.
- **Mentor card** — единый формат: фото + имя + chip "досвід N років" + направления.
- **Длина лендинга под тип продукта:** free ~8000px, paid ~12000px, event ~9000px.
- **Timer только для реальных дедлайнов** (иначе devalue).

## Когда расширять до v2
