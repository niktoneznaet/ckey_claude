# Copy patterns — формулы для CK-лендингов

Источник: KB копирайта CK (`14-landing-pages.md`, `01-brand-voice.md`, `10-offer-funnels.md`, `05-free-course-funnels.md`).

## 1. H1 формулы (по типу продукта)

### Paid course
- **"[Результат] через [срок], [механика]"**
  - "Стань Full-Time Trader за 4 місяці практики під наглядом менторів"
- **"[Глагол] у [сфера] і [результат]"**
  - "Торгуй crypto і forex під наглядом менторів які зробили на трейдингу свій капітал"

### Free course
- **"Стань [результат], використовуючи [метод]"**
  - "Стань прибутковим трейдером, використовуючи нашу методику торгівлі на крипторинку"
- **"Отримай [інструмент] за [кількість уроків]"**
  - "Отримай просту систему торгівлі за 9 коротких уроків, навчись контролювати ризики"
- **"[Глагол] у [сфера], щоб [результат]"**
  - "Навчись торгувати на Forex без хаосу та збільшуй свій капітал на 4-5% щомісяця"

### Event
- **"[Іменник]. [Іменник]."** — editorial короткий удар
  - "Три комьюнити. Один OKX Summit."
- **"[Суперлатив]-подія [часової період]"**
  - "Найцікавіша трейдинг-подія року"

## 2. H1 subtitle формулы

- **"[Болі ЦА] → [Рішення]"**
  - "Ти дізнаєшся, які знання та навички потрібні для того, щоб укладати до 80% прибуткових угод"
- **"[Контекст], [чому саме зараз]"**
  - "Подія року у трейдингу, ради якої варто закрити термінал і поїхати у Варшаву"

## 3. Eyebrow (над H2)

Завжди UPPERCASE, 11-13px, letter-spacing 0.12em. Приклади:

- `ДЛЯ КОГО ЦЕЙ КУРС` — benefits section
- `ПРОГРАМА` — curriculum
- `ФОРМАТ` — how it works
- `МЕНТОРИ` / `СПІКЕРИ` — team
- `РЕЗУЛЬТАТИ` — cases
- `ДОЛУЧАЙСЯ` — CTA
- `FAQ` — FAQ
- `НАВІЩО ДИВИТИСЬ` — event
- `КВИТКИ` — event tickets

## 4. CTA тексти

Порядок за силою:

1. **`Отримати доступ до уроків ↗`** — free course primary
2. **`Записатись на курс ↗`** — paid course primary
3. **`Купити квиток ↗`** — event primary
4. **`Почати безкоштовно ↗`** — lead magnet
5. **`Залишити заявку ↗`** — paid course (если ценник высокий)
6. **`Забрати бонус ↗`** — lead magnet с bonus
7. **`Підписатись на розсилку ↗`** — soft CTA

Secondary (ghost buttons):
- `Подивитись програму`
- `Дізнатись більше`
- `Безкоштовно`

**Правило:** CTA — це дія від першої особи ("Отримати доступ"), не інструкція ("Натисни тут").

## 5. Formula для benefits (для кого этот курс)

**4-5 пунктов:** "Якщо ти хочеш [дія + конкретний результат]":
- Перейти з найманої роботи на трейдинг як повноцінну професію
- Заробляти додатково $1000/міс паралельно з роботою
- Мати систему 4-5%/міс без стресу, зливів, вигорання
- Працювати онлайн з будь-якої точки світу
- Розібратись як реально працюють гроші та ринки

## 6. До / Після таблиця

| До | Після |
|----|-------|
| Не розумію графік | Бачу тренд, рівні, точки входу |
| Торгую навмання | Маю робочу модель торгівлі |
| Боюся втратити все | Знаю точно чим ризикую |
| Не знаю коли входити | Маю чіткі правила входу та виходу |
| Емоції керують | Дотримуюсь торгового плану |

## 7. Mentor bio шаблон

```
[Спеціалізація / фокус]. [Роки в трейдингу].
Команда Cryptology Key – на ринку з [рік].
```

Приклади:
- "Довгостроковий і середньостроковий контекст із фокусом на свінг-угоди. 6 років у трейдингу. Команда CK — на ринку з 2021 року." (Garry)
- "Price Action, структура ринку, ордер-блоки. 4 роки. Команда CK — з 2022." (Solva)

**Chips:** "Досвід N років" + "Forex/Crypto/Stock Market" + "Smart Money / Price Action / Market Profile"

## 8. FAQ canonical questions

**Paid course:**
1. Скільки часу треба на навчання?
2. Чи треба стартовий капітал?
3. Що якщо не розумію нічого в трейдингу?
4. Чи є розстрочка?
5. Скільки реально заробляють студенти?
6. Як відбувається навчання?
7. Що якщо не встигаю у групі?

**Free course:**
1. Для кого цей мінікурс?
2. Чи потрібен досвід у трейдингу?
3. Що входить до курсу?
4. Що робити після курсу?
5. Як спілкуються учасники?
6. Де навчатись після мінікурсу?

**Event:**
1. Скільки всього місць на саміт?
2. Як отримати знижку $50?
3. Буде онлайн-трансляція?
4. Де саме пройде івент?
5. Що входить у VIP-білет?
6. Як отримати білет безплатно?

## 9. Proof-numbers (используем везде)

Реальные цифры CK (из memory + `14-landing-pages.md`):
- **15K+ випускників** (було 12K на cryptology.school)
- **1000+ funded акаунтів**
- **250K комʼюніті** (сума соцмереж)
- **4+ роки на ринку**
- **30+ практикуючих трейдерів** у команді
- **4.8 рейтинг** (3К відгуків на edu-курс)
- **$240K+ максимальна виплата** (Gabak)

**Правило:** в hero даём 3 цифри (макс), не 7.

## 10. Запрещённые фразы (из 01-brand-voice.md)

- "Уважаемые слушатели / дорогие друзья"
- "Вы заработаете $X" (обіцянки конкретних сум)
- "Легкие деньги"
- "Не финансовый совет" — писать в FAQ, не в заголовках
- "100% гарантия результата"
- Критика конкретных конкурентов
- "Возьми кредит чтобы пройти курс"

## 11. Урочные номера (free course)

Кожен урок: `Урок N · HH хв · +1 методичка/роадмап`.

Шаблоны заголовків уроков (по структуре 14-landing-pages.md FTT):
1. "Чому трейдинг — найкраща професія" / "TradingView — твоя лабораторія"
2. "Заробіток на крипторинку" / "Читаємо графік за 10 хвилин"
3. "Шлях від новачка до профі" / "Біржа OKX + деморахунок"
4. "Ризик-менеджмент" / "Золоте правило 1%"
5. "Освіта та стартовий капітал" / "Математика прибутку: WinRate + RR"

## 12. Event-specific патерни

**"17 профі трейдерів"** — number + typology (из NeKonfa):
- "4 ГОДИНИ / 5 ДИСКУСІЙ / 12 РОЗІГРАШІВ"
- "3000+ учасників"
- "11 призів + грант"

**Schedule-line format:**
`[15:00] [ДИСКУСІЯ] | [Title, 1-2 рядки] | [Speakers]`

**Ticket tier naming:**
- `🐣 WAVE 1` — early birds, 100 мест
- `🎯 WAVE 2` — main wave, 400 мест  
- `👑 VIP` — премиум, $0 для OKX VIP-users
