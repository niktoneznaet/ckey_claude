# FTT anatomy — эталон paid course

Full-time Trader — флагман CK. 1920×19205px, 18 секций. Разобрано по слайсам сессии 028.

## Секции по порядку

1. **Nav (sticky)** — лого CK, меню (Про курс / Результати / Програма / Ментори / Відгуки / Тест), CTA-pill "Залишити заявку" справа.

2. **Hero** — massive stencil display "FULL-TIME TRADER" слева, справа phone-mockup с графиком $32,124. Chip "Оновлено у 2026 | Партнер OKX". CTA "Записатись на курс ↗". Нижче — 5 аватарок менторов + stats ($49K, 4.8, 15K+).

3. **Trust strip** — скроллинговая лента: "Торгуй CRYPTO і FOREX під наглядом менторів які зробили на трейдингу свій капітал закривай перші угоди вже через 4 тижні".

4. **4 features** — иконки в ряд: "Розбір ринку / Впевненість в діях / Свій метод / Стабільний дохід".

5. **Top results** — "ТОПОВІ РЕЗУЛЬТАТИ НАШИХ СТУДЕНТІВ" — 6 скриншотов выплат в 2 ряда: $240,152, $102,000, $100,000, $68,000, $50,750, $35,000. Каждый с датой, иконкой пропа, timestamp.

6. **"Твій шлях з нуля"** — **iso-3D воксельная анимация** 4 комнаты: 01 Підготовка / 02 Вибір ринку / 03 Практика / 04 Комʼюніті. Уникальный signature-asset FTT.

7. **"Твій тиждень на курсі"** — таблица 7 дней × 4 слота (ранок/день/вечір/ніч). Нагрузка по часам.

8. **Program table 4 tiers** — Junior / Middle / Senior / Supercharged. Checkmark-grid. Header цветной per tier. Самая плотная секция по data.

9. **"Сучасна платформа"** — phone-mockup + grid иконок (2×3): Кабінет / Lessons / Аналітика / Метрики / Щоденник трейдера / Ком'юніті.

10. **Mentors (3 cards)** — D-TRADER / GARRY / ALEKSA. Большое фото, chip досвіду, short bio. Фирменный паттерн.

11. **OKX partner strip** — скриншот терміналу OKX + badge "Офіційний партнер Cryptology Key".

12. **"Реальні історії"** — 4 mobile-screenshots в сетке с отзывами студентов.

13. **"Після курсу"** — 3 блока: проп-фірма / менторство CK / фонд управління активів.

14. **"Який ринок тобі підходить?"** — 4 цветных тайла: CRYPTO / FOREX / STOCK MARKET / COMMODITIES.

15. **CTA + iso-room** — "ГОТОВИЙ СТАТИ FULL-TIME TRADER?" + iso-сцена. Блок формы заявки.

16. **FAQ accordion** — "ЩЕ ТРОХИ ПРО КУРС" — 8-10 вопросов. Collapsed by default, open первый.

17. **Second CTA** — дублирует hero CTA.

18. **Footer** — полный: лого + Корисне / Контакти / Соцмережі / Документи. Копирайт, оферта.

## Цвета

- BG: `#0A0A0A` + голубой gradient glow в hero
- Accent primary: `#4A9EFF` (электрик-блю)
- Accent secondary: `#FF8A3D` (оранж) — для highlights в таблице и stats
- Chip text/labels: `#FFFFFF` / `#8B8B8B`

## Типографика

- Display: Unbounded ExtraBold/Black — весь hero и секционные H2
- Body: Manrope 400-600
- Program table — Manrope 500 для ячеек, Unbounded 700 для headers

## Что забрать в шаблон

- Program-таблица 4 тарифа — центральная фишка
- Mentor cards в ряд 3
- Stats row в hero
- CTA со стрелкой ↗
- Плотная секция Results с $$$-badges

## Что НЕ тащить (FTT-специфика)

- Iso-3D воксельные комнаты (требует отдельной иллюстрации)
- Таблица расписания "твій тиждень" — слишком кастомная
- Секция "Який ринок тобі підходить" — только для FTT
