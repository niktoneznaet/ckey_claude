# Создание сценария урока для мини-курса

## Задача

Создай сценарий видеоурока для бесплатного мини-курса по трейдингу. Следуй методологии из CLAUDE.md (шаблон урока, лестница роста, правила сценария).

## Входные данные от пользователя

Запроси у пользователя (если не указано):

1. **Название курса** и общая тема
2. **Номер урока** и его позиция (первый / середина / финальный)
3. **Тема урока** — о чём конкретно
4. **Цель урока** — что ученик должен понять / сделать / уметь после
5. **Кто ведёт** — какой эксперт (Garry, Dtrader, Solva, другой)
6. **Целевая длительность** — 5–15 минут

## Алгоритм

### Шаг 1 — Определи позицию в курсе

- **Первый урок:** Продаём прохождение курса. Карта курса + быстрая победа + предвкушение.
- **Середина:** Ценность. Чередуй простое и сложное. Информация + мотивация + действие.
- **Финальный:** Эмоции. Резюме трансформации + честный разговор + мягкий переход к анкете.

### Шаг 2 — Декомпозиция

Ответь на три вопроса:
1. Что нужно **знать**?
2. Что нужно **сделать**?
3. Что нужно **уметь** после?

### Шаг 3 — Напиши сценарий

Структура:

```
[ХУК] — 15–30 сек
Зацепи внимание. Вопрос, провокация, обещание результата.

[КОНТЕНТ УРОКА]
→ Информация: теория, концепция, правило — коротко и по делу
→ Мотивация: примеры, истории, последствия незнания, выгоды
→ Трансформация: конкретное упражнение, техника, задание

[ЗАКРЫТИЕ] — 30–60 сек
- Резюме: что узнал, что теперь умеет
- Превью следующего урока (кроме финального)
- CTA: выполни задание / переходи к следующему уроку
```

### Шаг 4 — Проверь по чеклисту

- [ ] Один урок = одна идея
- [ ] Хук в первые 15–30 секунд
- [ ] Смена формата каждые 2–3 минуты
- [ ] Конкретика вместо абстракций
- [ ] Есть практическое задание
- [ ] Язык понятен новичку
- [ ] Длительность 5–15 минут
- [ ] Есть эмоциональный якорь (история / метафора)
- [ ] Если первый урок — есть быстрая победа
- [ ] Если финальный — есть мягкий переход к анкете

## Формат выхода

Выдай сценарий в формате текста для суфлёра: разговорный стиль, короткие предложения, пометки [ПАУЗА], [ПОКАЗАТЬ ЭКРАН], [ПЕРЕКЛЮЧИТЬ СЛАЙД] где нужно. Укажи примерный хронометраж каждого блока.
