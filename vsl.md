## Структура VSL на основе анализа

### 1. ОТКРЫТИЕ (0:00-2:00)

**Формула:**
```
"В этом видео мы разберём полную схему как [РЕЗУЛЬТАТ] 
за [СРОК] с максимальной вероятностью даже если у вас нет [БАРЬЕРЫ]"

+ Представление: "Меня зовут [ИМЯ] и в [ВОЗРАСТ] я [ДОСТИЖЕНИЕ]"
+ Контраст: "Хотя ещё в [ВОЗРАСТ] я [ПЛОХАЯ СИТУАЦИЯ]"
```

### 2. ПРАВДА О РЫНКЕ (2:00-5:00)

**Паттерн разбора неправильных моделей:**
```
Модель X:
- Звучит привлекательно
- НО есть нюанс: [4 проблемы]
- Расчёт реальной прибыли (разочаровывающий)
- Личный опыт
- ВЕРДИКТ: не подходит
```

### 3. ПОШАГОВАЯ СИСТЕМА (5:00-40:00)

**Ядро — 5-7 шагов. Структура каждого:**
```
ШАГ N: [НАЗВАНИЕ]
- Критерии/правила (2-3 параметра)
- Конкретные примеры (10-15 штук)
- Ключевой инсайт
- Уровни/градация (от простого к максимальному)
- Личная история с цифрами
- 2-3 кейса учеников
- Переход к следующему шагу
```

**Тайминг:** Шаги по 4-9 минут каждый.

### 4. СОЦИАЛЬНОЕ ДОКАЗАТЕЛЬСТВО (по всему видео)

**Типы кейсов:** молодые (14 лет), пожилые (69 лет), мамы в декрете, люди без опыта, топовые блогеры.

**Формат:** короткие видео 30-60 сек, конкретные цифры, история до-после.

### 5. ПРЕЗЕНТАЦИЯ ПРОДУКТА (47:00-56:00)

```
А) Что получаешь: схема + шаблоны + поддержка
Б) Модули (каждый с результатом)
В) Бонусы (10-15 пунктов)
Г) Коммьюнити
```

### 6. ЗАКРЫТИЕ (56:00-60:00)

```
- Частота запуска (редко)
- Лист ожидания (без обязательств)
- Бонусы за заявку
- Финальная трансформационная история
```

### 7. ПОСТСКРИПТУМ

30-40 коротких отзывов подряд + финальный мега-кейс.

---

## КЛЮЧЕВЫЕ ПРИНЦИПЫ

1. **Конкретика:** точные цифры, сроки, примеры
2. **Контрасты:** было-стало, неправильно-правильно, мало-много
3. **Развенчание возражений** через кейсы
4. **Истории:** личные + учеников + известных людей
5. **Паттерн проблема-решение:** не работает → почему → правильное решение → доказательство
6. **Эмоциональные триггеры:** жадность, страх упустить, социальное доказательство, простота, срочность
7. **Повторение:** ключевые месседжи 3-5 раз разными словами

## ФОРМУЛЫ

**Хук:**
```
"Полная схема как [РЕЗУЛЬТАТ] за [СРОК] 
даже если у вас нет [БАРЬЕР 1], [БАРЬЕР 2] и [БАРЬЕР 3]"
```

**Контраст:**
```
"В [ВОЗРАСТ 1] я [ДОСТИЖЕНИЕ].
Хотя в [ВОЗРАСТ 2] я [УЖАСНАЯ СИТУАЦИЯ]"
```

**Неправильный подход:**
```
"[МОДЕЛЬ] — выглядит привлекательно. Но есть нюанс:
[проблемы] → [разочаровывающие цифры] → ВЕРДИКТ: не подходит"
```

**Шаг:**
```
"У [ОБЪЕКТ] есть [N] критерия → Примеры → Ключевой инсайт → Реальная история"
```

**Закрытие:**
```
"Делаю [ПРОДУКТ] раз в [СРОК]. Сейчас купить нельзя.
Лист ожидания: [преимущества]. Ни к чему не обязывает."
```
