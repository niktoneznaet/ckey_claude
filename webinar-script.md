# Сценарий вебинара / прямого эфира

> **Инсайты (воронки, вебинары, тон CK):** `Бизнес OS/knowledge-base/data/browser-claude-insights/воронки-и-вебинары.md`

## Задача

Создай сценарий вебинара или прямого эфира с чёткой структурой, хронометражем и переходами между спикерами. Цель — дать ценность + мягко перевести на анкету FTT.

## Входные данные от пользователя

Запроси (если не указано):

1. **Тема эфира**
2. **Аудитория** — студенты FTT / аудитория бесплатного курса / холодная
3. **Кто ведёт** — Влад + Гарри (диалог) / Гарри соло / другой формат
4. **Длительность** — оптимально 60–90 минут
5. **Цель финала** — анкета FTT / подписка на курс / чисто ценность
6. **Источники контента** — транскрибации, темы, материалы (если есть)

## Структура эфира (проверенная)

```
ОТКРЫТИЕ (5–7 мин) — Влад
├── Опрос в чат (вовлечение с первых секунд)
├── Обозначение темы (зачем пришли)
└── Быстрое представление

БЛОК 1 (15–20 мин) — Основной спикер
├── Ключевая проблема / разрушение иллюзий
├── Конкретика: цифры, статистика, бизнес-модель
└── Задание в чат

БЛОК 2 (15–20 мин) — Диалог
├── Углубление: почему проблема глубже чем кажется
├── Личная история / кейс
└── Задание в чат (другой формат)

БЛОК 3 (15–20 мин) — Основной спикер
├── Конкретный чеклист / шаги / решение
├── Практическое задание
└── [ПЕРВОЕ УПОМИНАНИЕ АНКЕТЫ — мягко, через самодиагностику]

БЛОК 4 (10–15 мин) — Диалог
├── Мотивация / масштаб возможности
├── Переход к FTT (если есть)
└── [ФИНАЛЬНАЯ АНКЕТА — через презентацию]

ЗАКРЫТИЕ (3–5 мин) — Влад
├── Резюме: три главных вывода
├── CTA: ссылка на анкету
└── Благодарность
```

## Правила

### Структура и тайминг
- Максимум 90 минут. Лучше 70 и закончить раньше, чем 2.5 часа.
- Разогрев — максимум 5 минут. Без обсуждения звука, лампы, Counter-Strike.
- Тема заявлена = тема раскрыта. Не размазывать.

### Чередование
- Негатив (проблема) → Конструктив (решение) → Мотивация
- Не два «тяжёлых» блока подряд
- Переключение между спикерами каждые 5–7 минут

### Вовлечение чата
- Минимум 3 задания/вопроса за эфир
- Разные форматы: число, слово, чеклист, открытый вопрос
- Вопросы из чата — только по теме текущего блока. Остальное: «разберём отдельно»

### Продажа
- Ссылка на анкету — максимум 2 раза (после блока 3 + финал)
- Каждый раз разными словами, через разный угол
- Продающий блок — максимум 5–7 минут. Детали — на звонке отдела заботы.
- НЕ повторять одну и ту же рекламную вставку 8 раз

### Переход к FTT (если нужен)
1. Самодиагностика: 3 вопроса, пусть ответят себе
2. Мост: «вы идёте одни, а можно с командой»
3. Спикер закрывает 3 боли аудитории
4. Презентация как путь, не как продукт
5. Честное закрытие: «анкета ни к чему не обязывает»

## Формат выхода

Полный конспект эфира с:
- Памяткой перед эфиром (кто что делает, на что обращать внимание)
- Текстом для каждого блока (ключевые фразы, переходы, вопросы в чат)
- Хронометражем
- Пометками: [ВЛАД], [ГАРРИ], [ОПРОС В ЧАТ], [ПОКАЗАТЬ СЛАЙД], [ССЫЛКА НА АНКЕТУ]

Формулировки и аналогии должны быть оригинальными. Если используются чужие материалы как источник — переработать так, чтобы идеи были те же, но подача, примеры и метафоры свежие.
