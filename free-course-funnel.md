## Задача

Создай полную текстовую воронку бесплатного курса CK — от велкама до пост-курсового дожима. Все сообщения для всех каналов, по дням, с сегментацией и триггерами.

Это НЕ отдельные сообщения (для этого есть /funnel-messages), а **полная структура воронки** — день за днём, канал за каналом.

## Входные данные

Запроси (если не указано):

1. **Название курса** — как называется
2. **Кол-во уроков** — сколько штук
3. **Темы уроков** — краткое описание каждого (1 строка на урок)
4. **Спикер(ы)** — кто ведёт (Garry, Solva, D-Trader, Nana, GZK)
5. **Целевой платный продукт** — куда ведём после курса:
   - `FTT` — Full-Time Trader (флагман, 2+2 мес, теория+практика)
   - `Workshop` — повышение квалификации (11 нед)
   - `Alpha Trader` — психология торговли (Nana, 8 модулей)
   - `без оффера` — только контент, CTA на соцсети
6. **Бонус-пак за email** — что даём за почту (Notion-шаблон, журнал сделок, чек-лист, стратегия)
7. **Скидка/оффер** — если есть скидка на платный продукт (%, дедлайн)
8. **Языки** — UA / RU / оба (по умолчанию: оба)

## Архитектура воронки — 5 блоков

Генерируй ВСЕ блоки последовательно. Каждое сообщение — в формате MSG-XXX.

---

### БЛОК A: ВХОД (День 0, сразу после регистрации)

#### MSG-001: Велкам | TG бот | День 0, +0 мин

Первое сообщение бота. Структура:
```
[Привет от спикера — 1 строка, тёплая, неформальная]

[Что тебя ждёт в курсе — 3 ключевых результата буллетами]

[Ссылка на платформу / первый урок]
```

Тон: как сообщение от друга, который реально рад что ты пришёл. Без пафоса.

#### MSG-002: Интро-видео | TG бот | День 0, +10 сек

```
[Короткое сообщение: все N уроков готовы на платформе]
[CTA: начать с урока 0 / урока 1]
```

#### MSG-003: Соцсети + что получишь | TG бот | День 0, +20 сек

```
[N ключевых навыков/итогов курса — нумерованный список]
[Подписки на соцсети CK (TG канал, Instagram, YouTube)]
```

---

### БЛОК B: УРОКИ (День 0 → День N)

Для **каждого урока** генерируй пуш-сообщение.

#### MSG: Пуш на урок N | TG бот | +2ч после предыдущего (если не смотрел)

Формула каждого пуша:
```
[Что ты узнал/сделал в прошлом уроке — 1 строка, конкретный результат]

[Что будет в этом уроке — 1-2 строки, интрига или конкретный навык]

[▶️ Урок N]
```

**Правила пушей:**
- Каждый пуш РАЗНЫЙ — не повторять формулировки
- Ссылаться на конкретный результат предыдущего урока
- Прогрессия: от базы к системе ("теперь ты знаешь X, следующий шаг — Y")
- Не слать 22:00–10:00 (отложить на 10:00)

#### Кружки спикеров в канал (2-3 точки за курс)

На ключевых точках (~40% и ~80% курса) — ТЗ на кружок спикера:

```
[КРУЖОК: спикер, ~40% курса]
Посыл: мотивация + "ты уже знаешь больше чем 90% новичков" + тизер следующего блока
Длительность: 30-60 сек
```

```
[КРУЖОК: спикер, ~80% курса]
Посыл: "финишная прямая" + подготовка к практике + эмоция гордости
Длительность: 30-60 сек
```

### БЛОК C: СБОР EMAIL + CTA НА ПЛАТНЫЙ

#### MSG: Сбор email | TG бот | После урока ~3 (настройка инструментов)

Два варианта — сегментация по наличию email:

**Вариант A (нет email):**
```
[Контекст: ты настроил инструменты, пора получить бонусы]

[Что в бонус-паке — 3 пункта]

[CTA: напиши свой email прямо сюда — отправим бонусы на почту]
```

**Вариант B (есть email):**
```
[Тот же контекст, но без запроса email]
[Бонусы уже отправлены / ссылка на Notion]
```

#### MSG: Email с бонусами | Email | +0 мин после ввода email

```
Subject (UA): [конкретный бенефит, 30-50 символов, без спам-триггеров]
Preheader: [дополняет subject]

[Привіт!]
[Забирай стартовий пак трейдера — зроби копію та починай використовувати]

[Що всередині:
— [Бонус 1]
— [Бонус 2]
— [Бонус 3]]

[Кнопка: Забрати бонуси → ссылка на Notion]
```

#### MSG: A/B оффер на платный продукт | TG бот | После урока ~4-5 (ключевой навык), +0 мин

Три фрейма (A/B тест — генерируй все три):

**Фрейм 1 — Консультация:**
```
[Вопрос-развилка: "Крипто чи Форекс — з чого почнеш?"]
[Спикер рекомендует выбрать ОДИН рынок]
[Курс [ПРОДУКТ] даёт базу для обоих + менторинг]
[CTA: консультация 15 мин, бесплатно → заполни анкету]
[Оффер: -X% предзапись + N бонусов, дедлайн]
```

**Фрейм 2 — Результат:**
```
[Хук: "Демо — добре. Але заробити перші $500 реальних — інше відчуття"]
[Проблема: для профі торгівлі потрібен ментор + стратегія + психологія]
[Рішення: курс [ПРОДУКТ] дає все це]
[CTA: анкета → консультація 15 хв]
[Оффер: -X% + бонуси]
```

**Фрейм 3 — Бонус:**
```
[Прямой оффер с акцентом на бонусы]
[Список бонусов предзаписи]
[CTA: анкета → консультація]
[Дедлайн]
```

### БЛОК D: РЕЕНГЕЙДЖ (для тех, кто перестал смотреть уроки)

Параллельная цепочка — запускается если юзер не открывал уроки.

#### MSG: X1 | TG бот | +12ч без активности

```
[Мягкий пуш — "дай 10 хвилин новому уроку"]
[Не давить, не обвинять]
[Кнопка: Продовжити курс]
```

#### MSG: X2 | TG бот | +24ч без активности

```
[Выгоды трейдинга — 3 пункта:]
[1) додатковий дохід без звільнення]
[2) $300+ зі старту навіть без великого капіталу]
[3) хобі — торгуй коли є час і настрій]
[Кнопка: Продовжити курс]
```
Не слать 22:00–10:00 → отложить на 10:00.

#### MSG: X3 | TG бот | +36ч без активности

```
[Доступ назавжди — повертайся коли натхне]
[А поки підпишись на соцсети — там теж корисне]
[Посилання: TG канал, Instagram, YouTube]
[Кнопка: Продовжити курс]
```

#### MSG: X4 | TG бот | +48ч без активности

```
[Кросс-продажа на другие бесплатные курсы CK]
[Може інший формат більше зайде?]
[Список: 2-3 альтернативных бесплатных курса с коротким описанием]
[Кнопка на каждый курс]
```

---

### БЛОК E: ПОСТ-КУРСОВОЙ (День N+1 → N+10)

#### MSG: Поздравление | TG бот | Сразу после последнего урока

```
[Вітаю! Ти зробив те, що більшість навіть не починає]
[Короткий итог: что теперь умеешь (3-4 навыка)]
[Але це лише початок — для стабільного результату потрібен ментор + практика + зворотний зв'язок]
[Мягкий CTA на платный продукт или "що далі"]
```

#### MSG: VSL-серия | Email | +48ч после завершения, каждые 48ч (5 писем)

**VSL-1: Проблема + интрига**
```
Subject: [от первого лица спикера, боль аудитории]

[Від першої особи спікера]
[Біль аудиторії — конкретика]
[Тизер: "записав N-хвилинне відео, де розбираю..."]
[CTA: посилання на YouTube]
```

**VSL-2 (+48ч): Мифы и ошибки**
```
Subject: [3 ошибки / мифа]

[3 поширені помилки]
[Конкретний кейс на кожну]
[CTA: "дивись відео"]
[P.S. тизер кейса з наступного листа]
```

**VSL-3 (+48ч): Кейс трансформации**
```
Subject: [имя студента + результат]

[Історія конкретної людини: було → стало]
[3 конкретні зміни]
[CTA: відео + лендинг продукту]
```

**VSL-4 (+48ч): Система / пошаговый план**
```
Subject: [N шагов от нуля до результата]

[N кроків від нуля до результату]
[Частину розкрити, за рештою — у відео]
[CTA: відео]
```

**VSL-5 (+48ч): Дожим + бонус**
```
Subject: [последнее письмо / бонус]

["Останній лист" (м'яка терміновість)]
[Таймкоди відео]
[CTA: Instagram + TG-канал спікера + лендинг]
```

**Правила VSL email:**
- Subject: 30-50 символов, без спам-триггеров
- From: имя спикера (не "Cryptology Key")
- Язык email: UA (по гайду email-доставляемости)
- Короткие абзацы, 1 кнопка-CTA
- Не дублировать бота — email идёт параллельно, но с ДРУГИМ контентом

## Что НЕ генерирует этот скилл

- Скрипты видеоуроков (→ /lesson-script)
- Картинки / баннеры / мемы — только текст
- Лендинг курса
- Сценарий вебинара (→ /webinar-script)

## Тон и правила

- Из `01-brand-voice.md`: неформальный (2/10 формальности), микс RU/UA, без пафоса
- Каждый урок = конкретный навык, не "вода"
- Пуши прогрессивные: урок N ссылается на результат N-1
- CTA мягкий: "заповни анкету — розкажемо детальніше" (не "купи")
- Антипаттерн: НЕ обещать конкретных цифр дохода, НЕ "лёгкие деньги"
- Продажа через экспертизу: 70% контент → 30% нативная продажа
- Ночное окно: не слать 22:00–10:00
